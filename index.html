Get started

Running in two minutes

No Terminal. No complicated setup. Five steps.

1

Download

Click the download button. You get TrayeeIME-1.0.0.dmg

2

Install

Open the DMG. Drag TrayeeIME to your Applications folder.

3

Bypass security warning

If you see "Apple could not verify" — right-click TrayeeIME.app in Applications, click Open, then click Open again. Or run the included fix_security.command file.

4

Grant permission

System Settings, Privacy and Security, Accessibility. Toggle TrayeeIME on. One-time step.

5

Start typing

Press Cmd+Shift+D to activate. Type in any app. संस्कृतम् begins.
